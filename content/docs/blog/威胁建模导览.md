# 威胁建模导览

## STRIDE

| Threat      | Desired property           |
| ----------- | -------------------------- |
| Spoofing    | Authenticity               |
| Tampering   | Integrity                  |
| Repudiation | Non-repudiability          |
| Information | disclosure Confidentiality |
| Denial      | of Service Availability    |
| Elevation   | of Privilege Authorization |

| 威脅     | 理想屬性 |
| -------- | -------- |
| 欺騙     | 身份验证 |
| 篡改     | 完整性   |
| 否認     | 不可否認 |
| 資訊泄露 | 保密     |
| 阻斷服務 | 可用性   |
| 特权提升 | 授權     |
