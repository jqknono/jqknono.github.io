## 准备

### 决定控制平台

- 确认倾向的平台, 避免买错商品.
- 默认支持手机双平台, 苹果 HomeKit 除外.
- `\`表示未知.

| 平台     | 开发能力 | 性价比 | 支持设备 | 广告 | 隐私保护 |
| -------- | -------- | ------ | -------- | ---- | -------- |
| 米家     | 8        | 9      | 多       | 有   | 低       |
| HomeKit  | 8        | 3      | 多       | 无   | 高       |
| yeelight | \        | 5      | 仅自家   | \    | \        |
| 格力     | 1        | 5      | 仅自家   | 有   | 低       |
| 美的     | 3        | 7      | 仅自家   | 有   | 低       |
| 华为     | 7        | 4      | 偏少     | 少   | 中       |

### 决定购买品牌

设备支持的平台

| 品牌     | 性价比 | 可接入平台              |
| -------- | ------ | ----------------------- |
| 小米     | 高     | 米家                    |
| yeelight | 一般   | 米家, yeelight, HomeKit |
| 格力     | 一般   | 格力, 华为              |
| 美的     | 较高   | 美的, 华为              |
| 云米     | 高     | 米家                    |

### 控制网络

- Wifi, 绝大多数支持 WiFi 的设备只支持 2.4G 网络.
- 蓝牙, 不支持 WiFi 的设备需要搭配中央控制器--网关. 网关一般绑定平台, 按平台选购.
