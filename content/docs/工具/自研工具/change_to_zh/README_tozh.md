# README_tozh

Chrome 插件: 自动切换文档到中文
Edge 也可以使用.

## 如何使用

0. 下载本库
1. 浏览器进入 edge://extensions/ 或 chrome://extensions/
2. 打开"开发者模式"
3. 按钮"Load unpacked"或"加载已解压的扩展程序"
4. 选择本库文件夹
5. 打开扩展开关开始使用
