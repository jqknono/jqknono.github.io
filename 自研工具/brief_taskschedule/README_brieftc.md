# brieftc

Crontab generator for Windows

一个简洁的定时任务生成器.

创建一个定时任务时, Windows 提供的计划任务管理器需要点很多下鼠标, 尽管功能齐全, 但是配置繁琐, 此工具可以帮助我们简化配置.

当然, 此工具也会非常不齐全, 克制添加功能.

基于.NET Framework 4.8, 只支持 Win10/11, Windows Server 2022 以上系统
